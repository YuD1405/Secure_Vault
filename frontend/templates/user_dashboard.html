Welcome to Secure Vault

Secure Vault is a security platform that enables users to encrypt/decrypt files, digitally sign and verify documents, manage personal RSA keys, and perform secure multi-factor authentication (MFA).

Quick Navigation Guide

  • Dashboard: Overview and welcome screen.
  • Manage RSA Keys: Generate and manage your RSA key pairs securely.
  • Encrypt File: Encrypt a file for another user using their public key.
  • Decrypt File: Decrypt a file using your private key.
  • Sign File: Digitally sign any file with your private RSA key.
  • Verify Signature: Validate the signature of a file using stored public keys.
  • Generate & Scan QR Code: Create or scan QR codes that store public key metadata.
  • Update Account Info: Edit your profile details or change your passphrase.
  • Logout: Securely exit your session.
